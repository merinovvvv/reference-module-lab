## Шаг 1. Описание справочников

### Справочник 1: Категории приложений (`AppCategories`)

| Колонка       | Тип           | Описание                              |
|---------------|---------------|---------------------------------------|
| `id`          | INT, PK       | Уникальный идентификатор              |
| `name`        | TEXT          | Название категории                    |
| `description` | TEXT          | Описание категории (многострочное)    |
| `created_date`| DATE          | Дата добавления категории в App Store |
| `min_age`     | INT           | Минимальный возраст пользователя      |
| `avg_rating`  | DECIMAL(3,2)  | Средний рейтинг по категории (0–5)    |

---

### Справочник 2: Мобильные приложения (`MobileApps`)

| Колонка          | Тип           | Описание                                      |
|------------------|---------------|-----------------------------------------------|
| `id`             | INT, PK       | Уникальный идентификатор                      |
| `name`           | TEXT          | Название приложения                           |
| `developer`      | TEXT          | Имя разработчика или компании                 |
| `description`    | TEXT          | Описание приложения (многострочное)           |
| `release_date`   | DATE          | Дата первого релиза                           |
| `price`          | DECIMAL(8,2)  | Цена в USD (0.00 — бесплатно)                 |
| `downloads_count`| INT           | Количество загрузок                           |
| `rating`         | DECIMAL(3,2)  | Рейтинг приложения 0.00–5.00                  |
| `category_id`    | INT, FK       | Ссылка на `AppCategories.id`                  |

---

## Зависимость справочников

Поле `MobileApps.category_id` является внешним ключом и ссылается на `AppCategories.id`.  
Каждое приложение принадлежит ровно одной категории. Категория может существовать без приложений.  
При удалении записи из `AppCategories` записи в `MobileApps` **не удаляются**.

## Покрытие типов данных

| Тип данных            | Поля                                              |
|-----------------------|---------------------------------------------------|
| TEXT (однострочный)   | `name`, `developer`                               |
| TEXT (многострочный)  | `description`                                     |
| DATE                  | `created_date`, `release_date`                    |
| INT (целое)           | `min_age`, `downloads_count`                      |
| DECIMAL (с запятой)   | `avg_rating`, `price`, `rating`                   |
